## 项目简介

本项目基于**纽约市2023年1月黄色出租车行程公开数据集**（约300万条记录），构建了一个完整的**数据分析 + 预测建模 + 智能问答系统**。

系统能够通过自然语言提问，快速查询出行规律、区域热度、车费估算，并支持某区域某时段的出行需求预测。

---

## 项目功能模块

### M1 数据处理
- 加载 Parquet 格式的原始数据
- 生成数据质量报告（缺失率、异常值统计）
- 数据清洗（异常值过滤、缺失值处理）
- 特征工程：提取小时、星期、是否高峰期等时间特征，新增行程时长、平均速度等衍生特征

### M2 分析可视化
完成以下4项分析，并自动生成图表保存至 `outputs/` 目录：
1. 出行需求时间规律（分小时、工作日/周末折线图）
2. 区域热度分析（TOP10 Borough 柱状图 + 高峰期对比）
3. 车费影响因素分析（距离-车费散点图 + 回归线）
4. 自选分析：各 Borough 平均车费与平均速度对比

### M3 预测模型
- 使用 **TensorFlow/Keras** 构建多层感知机（MLP）神经网络
- 与 **Random Forest** 进行对比实验
- 评估指标：MAE、RMSE
- 绘制神经网络训练 Loss 曲线
- 结论：随机森林在本任务（小样本）上表现更优

### M4 智能问答接口
- 支持自然语言交互（命令行）
- 支持至少6种问题类型：
  - 时段订单查询（如“晚上8点订单多少”）
  - 区域热度排名（如“哪个区域最热门”）
  - 区域+时段需求预测（如“预测皇后区9点需求”）
  - 车费估算（如“10英里大概多少钱”）
  - 高峰期与周末差异对比
- 返回具体数值结论 + 对应图表路径
